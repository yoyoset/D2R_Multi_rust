# D2R Multiplay v2.6.0 更新指南 (中文版)

本文档旨在说明 v2.6.0 "便当盒 & 个性化" 版本的核心改动。

## 🎨 视觉与布局优化

### 1. 便当盒 (Bento) 仪表盘
*   **3 列网格**: 在普通窗口尺寸下，由原来的 2 列升级为 3 列。卡片比例更加协调（4:3），减少了屏幕空间的浪费。
*   **高对比度显示**: 战网 ID 和 备注 强制加粗（Bold），字号增大。即使在远程控制或小屏幕上，关键信息也清晰可辨。

### 2. 高密度水平列表
*   **水平排布**: 列表模式（List Mode）现在是真正的水平单行排布。
*   **布局顺序**: `[头像] [用户名] [战网 ID] [备注] [状态点]`。

## 🖼️ 个性化头像系统

### 1. 内置图标库
您可以直接从内置的 7 类 D2R 职业图标中选择，快速区分不同的账号角色。

### 2. 自定义图片上传
*   **操作**: 在账号编辑弹窗中点击 [✨] 按钮即可上传本地图片。
*   **原理**: 图片会自动转为 **Base64** 编码存入 `config.json`。
*   **优势**: 彻底解决了“软件换路径导致图片丢失”的问题，您的头像随配置文件走，实现真正的绿色便携。

## 🔃 自由拖拽排序
*   **功能**: 您可以通过鼠标直接拖拽账号卡片（或列表中的手柄）来调整它们在主界面上的显示顺序。
*   **保存**: 排序结果会自动保存到后端，下次启动应用时依然生效。

## 🧹 数据精简
*   **移除区域 (Region)**: 移除了传统的地区选择下拉框。事实证明，大多数玩家通过战网启动器或备注来管理地区，移除此字段后界面更加整洁。

---

## 🛠 安装与运行
*   **格式**: 本应用为绿色免安装 ZIP 包。
*   **主程序**: `D2R_Multiplay.exe`
*   **配置文件**: 保存于 `%APPDATA%/d2r-multiplay/config.json`。

**再次提醒**: 请务必使用 Windows 账号的 **原始登录密码**，Windows Hello/PIN 码由于系统 API 限制暂时无法直接支持。
