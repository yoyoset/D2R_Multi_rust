# 隐私政策

## 生效日期：2026年2月11日

D2R Multiplay (Rust Edition) 是一款用于管理多开游戏客户端的本地桌面应用程序。我们非常重视您的隐私，并致力于妥善保护。

## 1. 数据收集

我们**不会收集**任何个人身份信息 (PII)。应用程序管理的所有数据均**存储在您的本地计算机上**。

具体包括：

- **账户信息**：用户名和配置存储在公开的应用程序数据文件夹中。
- **登录凭据 (Tokens)**：游戏特定的 `product.db` 文件仅在本地进行轮换。我们不会将这些凭据传输到任何外部服务器。
- **日志**：应用程序日志存储在本地，仅用于调试目的。

## 2. 数据使用

本地数据仅用于实现多开功能，例如：

- 管理游戏客户端实例。
- 轮换账户特定的配置文件。

## 3. 数据安全

由于所有数据都存储在本地，您的数据安全取决于您个人计算机的安全。我们建议：

- 仅在必要时以管理员身份运行应用程序。
- 保护您的 `%APPDATA%` 文件夹。

## 4. 第三方服务

本应用程序未集成任何第三方分析、追踪或云存储服务。

## 5. 政策修订

我们可能会不时更新隐私政策。如有变更，我们将通过在本仓库发布新的隐私政策来通知您。

## 6. 联系我们

如果您对我们的隐私政策有任何疑问或建议，请随时通过 GitHub Issues 与我们联系。
