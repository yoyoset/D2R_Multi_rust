# D2R Multiplay v0.2.0 Release Notes

## 🚀 重大更新 / Major Highlights

### 1. 🔔 统一通知系统 (Unified Notification System)

引入了全新的 `NotificationManager`。系统中的核心决策（如账号删除、重要设置更改、启动冲突）现在都会通过**阻塞式通知**展示，确保用户不会错过任何重要提示，彻底杜绝了静默失败的情况。

### 2. 📑 全量日志追踪 (Atomic Log Tracing)

仪表盘现在集成了一个实时的日志面板。您可以观察到游戏启动序列的每一个原子步骤（Atomic Steps）。这不仅提升了系统透明度，也让排查启动问题变得异常简单。

### 3. 💾 仪表盘视图记忆 (Dashboard Persistence)

软件现在会记住您的审美偏好。无论您喜欢“卡片模式”还是“列表模式”，系统都会自动保存并在下次打开时完美复原。

### 4. 🎨 界面细节打磨 (UI Polish)

- **紧凑布局**: 账号编辑弹窗采用了同行标签设计，大幅减少了空间占用。
- **操作增强**: 列表模式下的按钮触控区经过优化，解决了鼠标穿透和误点的问题。
- **视觉平滑**: 去除了选中时的物理跳动效果，改为优雅的颜色过渡，视觉体验更加稳重。

## 🛠️ 改进列表 / Full Changelog

- **[新增]** 多语言标签系统 (`label_password`, `label_username`)。
