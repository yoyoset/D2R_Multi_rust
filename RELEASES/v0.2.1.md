# D2R Multiplay v0.2.1 Release Notes

## 🚀 自动化与分发升级 / Release & Distribution Upgrade

### 1. ⚡ 全自动流水线 (CI/CD Automation)

从 v0.2.1 开始，我们彻底告别了繁琐的本地手动编译。现在所有的版本都通过 GitHub Actions 云端自动构建、签名并发布。这确保了每一个二进制文件都是在纯净环境下生成的，且具备合法的数字签名，减少了安全软件的误报。

### 2. 🌍 静态分发与自动更新 (R2 & Static Updates)

我们将更新服务器迁移到了 **Cloudflare R2**。

- **更快的下载**: 全球 CDN 加速，下载安装包不再受限。
- **静默更新**: App 内的“自动更新”现在会直接从 R2 拉取经过签名的增量包，速度更快，成功率更高。

### 3. 📦 双版本选择 (Dual Distribution)

我们现在同时提供两种版本，满足不同用户的需求：

- **安装版 (Installer)**: 支持自动更新，适合长期固定使用的用户。
- **绿色版 (Portable)**: 解压即用，不写注册表，适合喜欢纯净环境或存放在 U 盘的用户。

## 🛠️ 改进列表 / Full Changelog

- **[新增]** “检查更新”功能现在会引导至官网 [squareuncle.com/download](https://squareuncle.com/download) 以获取更多下载选项。

- **[优化]** 规范化了项目目录结构，移除了所有冗余的调试文件和私钥，仓库更加精简。
- **[修复]** 修复了构建脚本在特定操作系统语言下无法识别版本号的 Bug。
